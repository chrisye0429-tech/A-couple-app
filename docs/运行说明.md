# 运行说明

## 当前工程

项目已初始化为 Expo + TypeScript 跨平台 App 工程。

主要命令：

- `npm install`：安装依赖。
- `npm run typecheck`：检查 TypeScript 类型。
- `npm start`：启动 Expo 开发服务。
- `npm run ios`：通过 Expo 启动 iOS 调试流程。
- `npm run android`：通过 Expo 启动 Android 调试流程。

## 当前验证结果

- 依赖已安装成功。
- TypeScript 类型检查已通过。
- 已安装 `@react-native-async-storage/async-storage`，用于本地保存手账页面结构。

## 本机建议

后续要稳定启动 Expo 开发服务，请在普通本机终端运行：

```bash
npm start
```

如果仍在 Codex 沙盒环境中启动，可以尝试：

```bash
npm run start:codex
```

普通本机终端通常直接使用 `npm start` 即可。
