# 首页与导航说明

## 当前页面结构

当前 App 使用轻量的本地状态完成基础导航，暂未引入复杂导航库。

底部导航包含四个入口：

- 首页：展示当前情侣空间概览、最近编辑手账和快速入口。
- 文件夹：展示手账文件夹列表和新建入口。
- 文件夹：支持新建、重命名和删除。
- 手账：展示可交互的本地手账编辑器，支持图片、文字、贴纸、拖拽、缩放、旋转、删除和保存。
- 共享：展示情侣邀请码、只读分享链接和权限状态。

## 设计原则

- 页面先保证清晰可用，不堆叠过多功能。
- 底部导航保持固定，方便新手理解。
- 首页展示真实产品状态，不做营销落地页。
- 手账编辑器优先保证核心编辑体验清晰可用，再逐步扩展更多信纸、字体、贴纸和同步能力。

## 后续演进

当页面数量增加、需要登录态或深层跳转时，再考虑接入正式导航库。首版当前阶段优先使用简单结构，减少依赖和复杂度。
