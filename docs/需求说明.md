# 需求说明

## 产品目标

做一款情侣共同使用的互动手账 App。情侣双方可以一起制作、整理、修改和分享属于两个人的手账页面，用轻松、清新、可爱的方式保存日常记忆。

## 核心用户

- 情侣双方：共同创建和编辑手账内容。
- 第三方访客：通过只读链接查看被分享的页面或文件夹。

## 首版核心功能

1. 手账页面编辑
   - 上传本地图片。
   - 添加和编辑文字。
   - 对文字进行基础排版。
   - 使用标准信纸背景。
   - 添加贴纸美化页面。
   - 图片、文字、贴纸可以在页面中自由拖拽排版。

2. 手账保存与整理
   - 完成后的手账页面可以保存。
   - 页面可以存放在文件夹中。
   - 文件夹可以命名和管理。

3. 情侣共享
   - 通过邀请码绑定情侣关系。
   - 情侣双方可以共享同一批手账内容。
   - 情侣双方可以共同修改手账页面。

4. 第三方只读分享
   - 可以生成分享链接。
   - 第三方打开链接后只能查看，不能修改。

5. 主题与视觉
   - UI 总体清新简洁。
   - 支持切换不同色系。
   - 切换色系不影响已编辑的手账内部内容。

## 首版暂不做

- 大型公开社区或陌生人社交。
- 复杂视频编辑。
- AI 自动生成整页手账。
- 商城、会员、付费贴纸系统。
- 复杂动画和高级 3D 效果。

## 成功标准

首版成功不是功能很多，而是用户可以稳定完成一个完整闭环：

注册或登录账号，绑定情侣关系，创建文件夹，制作一页手账，保存后双方都能看到并修改，再生成只读链接给别人查看。
