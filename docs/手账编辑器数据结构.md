# 手账编辑器数据结构

## 当前目标

本阶段已经进入本地手账编辑器原型优化：先保证单机编辑体验可用，暂不做云端同步。

当前代码位置：

- `src/features/journalEditor/types.ts`：编辑器核心类型。
- `src/features/journalEditor/mockData.ts`：本地示例手账页面数据。
- `src/features/journalEditor/EditorScreen.tsx`：编辑器主页面，负责基础状态、组件组装和数据传递。
- `src/features/journalEditor/EditorPanels.tsx`：编辑器独立面板组件，包含回退/前进、底部工具栏、背景模板面板、贴纸面板、文字编辑面板、图层列表和属性面板。
- `src/features/journalEditor/JournalCanvas.tsx`：手账画布组件，负责纸张背景、元素渲染、自由拖拽、双指缩放、四角操作点和对齐辅助线显示。
- `src/features/journalEditor/editorModel.ts`：编辑器纯数据逻辑，负责手机端页面尺寸适配、背景数据转换、自定义颜色校验、元素创建、元素复制和贴纸显示兜底。
- `src/features/journalEditor/editorAlignment.ts`：对齐辅助计算逻辑，负责画布边缘、画布中心、相邻元素边缘和相邻元素中心的吸附判断；默认不启用。
- `src/features/journalEditor/useEditorHistory.ts`：编辑器历史记录逻辑，负责撤销、前进和一次手势只记录一步。
- `src/features/journalEditor/useEditorPageActions.ts`：编辑器页面操作回调，负责选中、移动、缩放、旋转、复制、删除、文字编辑、背景切换、图片选择和本地保存。
- `src/features/journalEditor/storage.ts`：本机保存和读取手账页面结构。

## 页面数据

一页手账使用 `JournalPage` 表示，包含：

- 页面 id。
- 页面标题。
- 所属文件夹名称。
- 信纸背景信息。
- 页面尺寸。当前手机端信纸按竖版 4:3 比例设计，高宽对应为 400 x 300，基础尺寸为 300 x 400。
- 页面元素列表。
- 更新时间。
- 保存状态。

## 页面元素

页面元素统一使用 `JournalElement` 表示。首版包含三类：

- `image`：图片元素，当前支持从本地相册选择图片并放入画布。
- `text`：文字元素，当前支持编辑文字内容、字号、颜色、加粗和斜体。
- `sticker`：贴纸元素，当前支持从贴纸素材库中选择添加；画布内按透明 PNG 风格显示，不带默认底色。

每个元素都包含：

- id。
- 类型。
- 显示名称。
- 位置和尺寸。
- 旋转角度。
- 层级。
- 透明度。
- 是否锁定。

## 当前编辑器页面结构

编辑器页面分为四块：

- 顶部状态：显示页面标题、所属文件夹、保存状态。
- 手账画布：根据信纸和元素数据渲染页面。
- 历史操作：画布上方提供“回退 / 前进”按钮，支持撤销最近编辑并恢复刚才撤销的编辑。
- 预览滚动区：顶部状态、历史操作和手账画布在上方独立滚动区内，画布内容较高时只滚动预览区。
- 底部操作区：工具栏和当前工具对应的内容面板放在下方独立滚动区内；滚动底部面板时不会带动上方预览区。
- 工具栏：背景、文字、贴纸、分页、收藏、图片。
- 工具内容过滤：背景只显示背景模板；文字只显示文字内容、字体和样式；贴纸只显示贴纸分类和贴纸列表；图片只显示本地相册选择入口；收藏只显示收藏内容；分页只显示页面管理入口。
- 本地保存：点击“保存到本机”后，把当前页面结构写入本机存储；再次进入编辑器时会优先读取本机保存内容。
- 拖拽移动：按住画布上的图片、文字或贴纸即可移动元素，移动后会更新元素的 `frame.x` 和 `frame.y`，并将页面标记为本地草稿。
- 元素缩放：选中元素后可使用“缩小 / 放大”按钮调整 `frame.width` 和 `frame.height`，并自动限制在画布范围内。
- 双指缩放：在画布上用双指缩放选中元素时，会以元素中心为基准调整 `frame.width` 和 `frame.height`。
- 元素旋转：选中元素后可使用“左转 / 右转”按钮调整 `frame.rotation`。
- 叠加显示：页面元素按 `zIndex` 叠加渲染，点选元素时会自动提到上层，方便重叠后继续操作。
- 元素删除：选中图片、文字或贴纸后，可以在“元素管理”中删除。
- 四角操作点：选中元素后，元素四角会出现快捷操作；左上角删除，右上角旋转，左下角调整透明度，右下角复制一个相同元素。四角按钮只在圆形按钮附近响应触摸，元素中间区域保留给拖拽移动，避免小贴纸被四角热区覆盖后无法自由拖动。
- 对齐辅助：默认关闭。关闭时拖动元素会跟随手指自由移动；开启后才会识别画布边缘、画布中心和其他元素的边缘/中心，距离 6px 内吸附并显示横向或纵向辅助线。
- 文字编辑：选中文字元素后，可以在“文字内容”面板中修改多行文字，应用后更新元素的 `content`。
- 文字样式：选中文字元素后，可以调整 `fontSize`、`fontWeight`、`fontStyle`、`color` 和字体。切换字体时会先尝试下载并注册素材库字体；若远程字体暂时不可用，会使用本机替代字体，避免点击后外观完全不变。
- 背景模板：点击底部“背景”工具后，会显示 20 种手账背景模板，包含纯色、纹理和图案。
- 背景绘制：纯色背景直接用颜色绘制；网格、点阵、横线、牛皮纸、水彩、复古、碎花、波点、条纹、格子、云朵和星星使用本地组件绘制，不依赖网络图片。
- 自定义纯色：背景面板提供常用色板和 `#RRGGBB` 输入，用户可以应用自定义纯色背景。
- 贴纸素材：点击底部“贴纸”工具后，会显示素材库贴纸面板；贴纸按分类展示，每类读取本地 `stickers.json` 的前 20 个素材。点击贴纸后会在当前信纸中心附近新增，初始大小约为信纸宽度的 20%，并立即进入选中状态。
- 工具栏样式：底部工具栏采用“图标 + 标题”形式，便于新手快速识别背景、文字、贴纸、分页、收藏和图片入口。
- 旧数据适配：读取本机旧尺寸页面时，会迁移到当前 300 x 400 信纸尺寸，并按比例调整元素位置和大小。
- 撤销与前进：添加、删除、复制、移动、缩放、旋转、透明度、文字内容和文字样式修改都会进入本地历史记录。拖拽移动和双指缩放按“一次手势一步”记录，避免产生过多无意义历史。

## 本地保存机制

当前使用 `@react-native-async-storage/async-storage` 保存页面结构。

保存内容是完整的 `JournalPage`：

- 页面标题、文件夹、信纸。
- 所有页面元素。
- 元素位置、尺寸、旋转、层级和样式。
- 保存状态和更新时间。

当前本地保存只用于单机原型验证，不代表情侣双方云端同步。后续进入同步阶段时，可以把同一份 `JournalPage` 结构发送到后端。

## 后续开发方向

下一步可以基于当前结构继续推进：

- 添加新文字元素。当前已支持点击工具栏“文字”后，在画布中创建默认文字块，并同步到图层和属性面板。
- 添加新贴纸元素。当前已支持点击工具栏“贴纸”后，从爱心、五角星、萌宠、太阳中选择贴纸，并同步到图层和属性面板。
- 接入本地图片选择。当前已支持点击工具栏“图片”后打开系统相册，选中后把图片 URI 写入图片元素并显示到画布。
- 支持拖拽移动元素。当前已支持图片、文字、贴纸在画布内移动，移动后需要点击“保存到本机”持久保存。
- 支持元素缩放。当前已支持按钮式缩放和双指缩放图片、文字、贴纸。
- 支持元素旋转。当前已支持按钮式左转和右转。
- 支持删除元素。当前已支持删除选中的图片、文字或贴纸。
- 支持画布内四角快捷调整。当前已支持透明度、删除、复制和旋转。
- 文字元素当前只显示文字本身，不显示底色或边框。
- 支持文字内容编辑。当前已支持选中文字元素后通过编辑面板修改 `content`，并通过本地保存保留。
- 支持文字样式编辑。当前已支持修改字号、颜色、加粗、斜体和字体；选择字体时会调用素材管理器下载并注册字体。画布渲染自定义字体时不再强制叠加粗体，避免单文件字体被系统回退为默认字体。
- 支持 20 种背景模板。当前已支持 8 种纯色、6 种纹理和 6 种图案背景，并可在编辑器内切换。
- 支持自定义纯色背景。当前已支持常用色板和十六进制颜色输入。
- 支持可开关的对齐辅助。当前默认自由拖动；打开“对齐辅助”后，拖动时可对齐画布和相邻元素，辅助线在拖动结束后自动隐藏。
- 支持撤销和前进。当前已支持通过画布上方按钮回退最近编辑，并恢复刚才撤销的编辑。
- 支持素材库贴纸分类浏览。当前已把 `stickers.json` 的 20 个分类接入编辑器贴纸面板，新增贴纸不再只停留在配置文件中。
- 支持透明贴纸元素。当前贴纸元素优先使用可直接渲染的 PNG/WebP/JPG 图片地址；Iconify 等远程 SVG 地址暂时使用透明 emoji/文字兜底，避免在 React Native 图片组件中显示为空白。后续如接入 SVG 渲染库，可恢复 SVG 图片渲染。
- 将当前页面结构保存到本地存储。当前已支持点击“保存到本机”保存，并在进入编辑器时读取。
- 再进入情侣同步和只读分享。
